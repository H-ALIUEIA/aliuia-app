<h1>Λίγα λόγια</h1>

Έχετε βαρεθεί με της ειδήσεις και τα Social Media<br>
Δεν ξέρετε πλέον τι είναι προπαγάνδα και τη αλήθεια?<br>

Με την εξέλιξη της τεχνικής νοημοσύνης σε λίγο κανείς δεν θα ξέρει τη είναι η αλήθεια<br>

Θυμηθείτε μία εικόνα = 1000 λέξεις<br>
Άρα τα μάτια μας μπορούν να δουν 1000 ψέματα<br>

Ο μόνος τρόπος να βρείτε την αλήθεια είναι με δικιά σας έρευνα<br>
αλλά πως ξεκινάει κάποιος να το κάνει αυτό?<br>

Τώρα με αυτήν την εφαρμογή μπορείτε να ξεκινήσετε το ταξίδι σας στο να βρείτε την αλήθεια<br>

Μετά από χρόνια έρευνας μπορείτε να δείτε την δουλειά μου (μαζί με της πηγές) για να έχετε μία μεγάλη ώθηση στην ανακάλυψη της αλήθειας<br>

πολλά άτομα θα πούνε πως είναι πολύ δουλειά<br>
πολλά άτομα θα πούνε και τι θα γίνει άμα την βρω?<br>

Αλλά ως ένα άτομο που βρήκε την αλήθεια θα σας πω πως είναι αρκετά σημαντικό να την ξέρετε<br>
Γιατί η αλήθεια θα σας απελευθερώσει<br>

όλα τα προβλήματα της ζωής μου εξαφανίστηκαν μόλις κατάλαβα πως λειτουργούνε όλα(και ναι, εννοώ όλα)<br>
από στεναχώρια για το απελπιστικό μέλλον, την στεναχώρια του παρόντος, το συνεχές αρνητικό σκεπτικό μου κτλ.<br>

Ξέρω μπορεί να ακούγατε σαν μία ψεύτικη διαφήμιση αλλά δίνω όλη μου την έρευνα δωρεάν και δεν περιμένω να πάρω ούτε ένα ευρώ. Αυτό που λέω είναι απλά η πραγματικότητα όσο περίεργο και αν ακούγεται.<br>

Εννοείτε κάποιος δεν βρίσκει την αλήθεια σε μία μέρα αλλά άμα θέλετε να την βρείτε:<br>

Το πρώτο βήμα είναι να εγκαταστήσετε την εφαρμογή.<br>

Τι περιμένετε?<br>
